GitHub Sync
Deine Daten werden in einem GitHub-Repository als JSON gespeichert — kostenlos, privat, von überall abrufbar.
1
Gehe zu github.com/new und erstelle ein neues Repository (z.B. mindfulcart-data). Kann privat sein.
2
Gehe zu github.com/settings/tokens/new — wähle repo Scope — Token generieren und hier einfügen.
🔒 Token wird nur in deinem Browser gespeichert — nie an Dritte übertragen.
